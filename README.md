# Fretboard Pilot

A focused guitar reference app built with Next.js + TypeScript.

## What it includes

- chord library
- scale library
- neck view with position switching
- chord notes and intervals
- scale notes
- Tonal.js-backed data generation
- local validation CLI: `./bin/test-app.sh`

## Local development

```bash
npm install
npm run dev
```

## Test CLI

```bash
./bin/test-app.sh
```
